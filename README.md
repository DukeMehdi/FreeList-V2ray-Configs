# 📋 لیست کانفیگ های V2RAY رایگان 🆓

## ⚠️ راهنما:
لیستی از سرورهای رایگان و سالم v2ray با پشتیبانی از پروتکل‌های مختلف، فقط کافیه کپی کنی داخل نرم افزارهایی که لینکهاشو پایینتر واست میذارم بچسبونی و تموم یبار که اینکارو بکنی دیگه نیازی نیست مجدد بیایی اینجا و لینک رو کپی کنی! کانفیگها بروزرسانی شه لینک توام با زدن گزینه آپدیت اشتراکها یا همون آپدیت ساب توی نرم افزار مربوطه که دانلود کردی بروزر میشه، بدون دردسر.
**در آپدیت جدید کانفیگ ها هر سه ساعت یکبار آپدیت میشن بدلیل بررسی دقیقتر**

## ✅ نحوه استفاده:
لینک کانفیگ (`Trojan` ، `SSR`، `SS`، `Vmess` ، `VLess`) را کپی کن در برنامه های (مثل **Hiddify**، **v2ray**، **Happ**) وارد کن متصل شو و از اینترنت آزاد لذت ببر.
